تطبيق Kitchen/Chef App – إدارة المطبخ بسرعة ودقة

تطبيق Kitchen/Chef App مصمم لإدارة طلبات المطبخ بكفاءة، سواء للمطاعم أو المطابخ السحابية. التطبيق متكامل مع أنظمة الـ POS، وبيساعد الشيفات وفريق المطبخ يجهزوا الطلبات بدقة وسرعة.

تسجيل الدخول

أول خطوة هي تسجيل الدخول باستخدام الإيميل أو رقم الموبايل.

الصفحة الرئيسية

لوحة تحكم المطبخ تعرض الطلبات الجديدة فور وصولها:

  • طلبات من الزباين أونلاين، ترابيزات، أو تيك أواي.
  • تفاصيل الطلب زي نوع الأكل، الكمية، وملاحظات مثل "بدون بصل".

تحديث حالة الطلب

إدارة حالة الطلبات بسهولة:

  • تغيير الحالة إلى "قيد التحضير" أو "جاهز".
  • إشعار فوري لفريق التوصيل أو الجرسونات عند اكتمال الطلب.
  • مثال: تضغط "جاهز" بعد تحضير طلب بيتزا ليتم نقله للتوصيل.

قايمة الطلبات

عرض كل الطلبات الواردة للمطبخ:

  • فرز الطلبات حسب الحالة (جديدة، قيد التحضير، جاهزة).
  • ترتيب الطلبات حسب الأولوية للطلبات العاجلة.
  • مثال: ترتب طلبات التوصيل السريع قبل طلبات الترابيزات.

تفاصيل الطلب

عرض تفاصيل كل طلب بدقة:

  • الأكل المطلوب، الإضافات (مثل صوص أو جبنة)، وملاحظات الزبون.
  • كتابة ملاحظات داخلية للشيفات.
  • التواصل مع الجرسونات لتعديل الطلب إذا لزم الأمر.
  • مثال: تشوف ملاحظة "ياريت بسرعة" وتكتب تعليمات للفريق.

الإشعارات

إشعارات فورية للبقاء على اطلاع:

  • إشعار بوصول طلب جديد.
  • تنبيهات بتأخير الطلبات أو المشاكل.
  • مثال: إشعار بطلب عاجل يحتاج تحضير فوري.

إدارة الوقت

تحسين كفاءة المطبخ من خلال تتبع الوقت:

  • متابعة وقت تحضير كل طلب.
  • حساب متوسط وقت التحضير لتحسين الأداء.
  • مثال: تلاحظ إن تحضير البيتزا بياخد 10 دقايق وتعدل العملية لتقليل الوقت.

التواصل الداخلي

تواصل سريع مع فريق العمل:

  • إرسال رسائل للجرسونات أو المدير (مثل "الجبنة خلصت").
  • التنسيق مع السواقين لتوصيل الطلبات.
  • مثال: تبعت رسالة للمدير إن مكون معين قرب يخلّص.

الإعدادات

تخصيص التطبيق حسب احتياجاتك:

  • اختيار اللغة (عربي أو إنجليزي).
  • تغيير حجم الخط أو الألوان لتكون مريحة للعين.
  • ترتيب عرض الطلبات حسب الوقت أو الأولوية.
  • مثال: تختار عرض الطلبات العاجلة أولاً.

الملف الشخصي

إدارة بياناتك الشخصية:

  • تعديل الاسم، رقم الموبايل، أو صورة البروفايل.
  • إضافة إشعارات مخصصة للطلبات.
  • مثال: تضيف إيميلك لتلقي إشعارات إضافية.

تغيير كلمة المرور

تغيير كلمة المرور بسهولة:

  • إدخال الباسوورد القديم والجديد.
  • طلب رابط إعادة تعيين عبر الإيميل أو SMS.
  • مثال: تغير الباسوورد لضمان أمان الحساب.

تسجيل الخروج

حماية حسابك:

  • زر لتسجيل الخروج لمنع الوصول غير المصرح به.
  • مثال: تضغط "تسجيل الخروج" بعد انتهاء ورديتك.

لماذا تطبيق Kitchen/Chef App؟

المميزات الرئيسية:

  • إدارة طلبات المطبخ بدقة وسرعة.
  • تكامل مع أنظمة الـ POS وتطبيقات FoodKing الأخرى.
  • إشعارات فورية وتواصل داخلي لتحسين التنسيق.
  • واجهة سهلة ومرنة تناسب المطابخ المزدحمة.
  • حماية بيانات الشيفات والمطعم.
