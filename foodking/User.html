تطبيق User App – طلب الأكل بسهولة وسرعة

تطبيق User App مصمم للزبائن عشان يقدروا يطلبوا الأكل أونلاين، يتابعوا طلباتهم، ويديروا حساباتهم بكل سهولة. التطبيق بيوفر تجربة سلسة من بداية اختيار المطعم لحد استلام الطلب.

تسجيل الدخول

أول خطوة لاستخدام التطبيق هي تسجيل الدخول باستخدام الإيميل أو رقم الموبايل.

الصفحة الرئيسية

الصفحة الرئيسية بتعرض كل اللي ممكن تحتاجه بسرعة:

  • المطاعم القريبة من موقعك.
  • الأكلات الشائعة أو اللي عليها تريند.
  • العروض والخصومات المتاحة.
  • شريط بحث للوصول لمطعم أو أكلة معينة.

القايمة (Menu)

منيو المطعم اللي اخترته بيظهر هنا:

  • اختيار الأكلات وإضافة/إزالة إضافات (مثل جبنة أو بصل).
  • عرض السعر والوقت المتوقع للتوصيل.

العروض (Offers)

قسم خاص بالخصومات والكوبونات:

  • عرض كل العروض المتاحة.
  • تفعيل العرض أو إدخال كود خصم.

طلباتي (My Orders)

متابعة كل طلباتك السابقة والحالية:

  • عرض حالة الطلب (قيد التجهيز، تم التوصيل).
  • تتبع وقت وصول الطلب.
  • التواصل مع المطعم في حالة وجود مشكلة.

تعديل الملف الشخصي

إدارة بياناتك الشخصية:

  • تغيير الاسم، رقم الموبايل، أو الصورة الشخصية.
  • إضافة الإيميل لتلقي الإشعارات.

الدردشة

التواصل مع المطعم أو خدمة العملاء:

  • إرسال رسالة مباشرة.
  • فتح تذكرة دعم لحل المشاكل.

العنوان

إدارة عناوين التوصيل:

  • إضافة أو تعديل عنوان جديد.
  • إضافة تفاصيل مثل "جنب السوبرماركت" لتسهيل التوصيل.

تغيير كلمة المرور

تغيير كلمة المرور بسهولة:

  • إدخال كلمة المرور القديمة والجديدة.
  • طلب رابط إعادة تعيين عبر الإيميل.

تسجيل الخروج

زر لتسجيل الخروج لحماية حسابك.

لماذا تطبيق User App؟

المميزات الرئيسية:

  • تجربة طلب سهلة وسريعة.
  • إدارة كاملة للطلبات والحساب الشخصي.
  • تكامل مع عروض وخصومات لتوفير المال.
  • واجهة بسيطة تناسب الجميع.
